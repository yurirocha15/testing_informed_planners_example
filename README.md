# testing_informed_planners_example

## Installation

```shell
$ make install
$ make build
```

## Running

 - Checkout `moveit` to the `feature/support-informed-trees` branch
 - [To run EIT*/EIRM*] Checkout `ompl` to `bugfix/fix-eit-star-tc`
 - Run:

```shell
$ roslaunch test_environment test_environment.launch
```
